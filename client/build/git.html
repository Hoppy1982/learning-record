Git Commands

This is a record of the git commands I've used with examples of how I've used them. The plan is to at least dump placeholders on this page which I can return to and read up on / flesh out later on. I wanted to start doing this as git seems such a broad topic but you can do some cool stuff without having to know each and every command inside out.

status

To see what is / isn't staged for commit on current branch.

$ git status

checkout

Switches between branches / commits.

$ git checkout branchName

diff

To see the differences between two branches.

$ git diff branchOne branchTwo

log

To view commit history. 'q' to exit from the log abck into bash.

'q' to exit from the log abck into bash.

Use this to get commit names if you need to checkout or revert them.

$ git log

Short pretty format

$ git log --pretty=short

remote

To see the remote url

$ git remote -v

add

To add all modified files to staging area

$ git add .

To add an individual file to staging area

$ git add /examplePath/exampleFile.js

commit

Creates a commit on current branch.

If the -m flag is not used default text editor will open.

$ git commit -m "this is a commit message"

Edit to message of the most recent commit.

$ git commit --ammend -m "edited commit message"

branch

To view all branches and current branch.

$ git branch

To create a new branch from the current branch.

$ git branch newBranchName

set-upstream

To set the remote for current branch.

$ git branch --set-upstream-to=origin/branchName

push

To push current branch up to the set remote.

$ git push

To set the remote and push to it.

$ git push -u origin branchName

To force push to the set remote. I've done this when the remote is has changes that would be lost but I don't care and want to overwrite the remote from local.
